## Run locally
1) Install Node.js 18+
2) In project folder:
   - `npm install`
   - `npm run dev`
3) Open the URL shown in terminal (usually http://localhost:5173)

## Edit details/content
- `src/data/site.json`
- `src/data/services.json`
- `src/data/gallery.json`
- `src/data/videos.json`

## Background video
A premium looping background video is included at:
- `public/bg.mp4`

To replace it, keep the filename `bg.mp4` (or update `backgroundVideo` in `src/data/site.json`).

## Logo
- `src/assets/logo.svg` (replace with your own SVG if needed)

## Deploy
- Build: `npm run build`
- Upload `dist/` to any static hosting (Netlify, Vercel static, GitHub Pages, etc.)
